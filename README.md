# Floci + S3 + SNS + Lambda Demo

## Step:-2

### Start Floci Container

Starts the Docker container for Floci and exposes local AWS-like services on your machine.

```bash
docker run --rm -p 4566:4566 \
  -v /var/run/docker.sock:/var/run/docker.sock \
  floci/floci:latest
```

## Step:-6

### Create AWS Resources on Floci

#### Create S3 Buckets

```bash
aws --endpoint-url=http://localhost:4566 s3 mb s3://app-logs-bucket-2
aws --endpoint-url=http://localhost:4566 s3 mb s3://processed-logs-bucket-2

# Verify buckets
aws --endpoint-url=http://localhost:4566 s3 ls
```

### Create SNS Topic

```bash
aws --endpoint-url=http://localhost:4566 sns create-topic --name log-alerts-topic-2
```

### Subscribe Email to SNS

```bash
aws --endpoint-url=http://localhost:4566 sns subscribe \
  --topic-arn arn:aws:sns:us-east-2:000000000000:log-alerts-topic \
  --protocol email \
  --notification-endpoint tandelsanjana8303@gmail.com
```

---

## Step:-7

### Deploy Lambda Function

```bash
aws --endpoint-url=http://localhost:4566 lambda create-function \
  --function-name log-processor-node-2 \
  --runtime nodejs20.x \
  --handler index.handler \
  --zip-file fileb://function.zip \
  --role arn:aws:iam::000000000000:role/lambda-role \
  --timeout 30 \
  --memory-size 256 \
  --environment Variables="{
    SNS_TOPIC_ARN=arn:aws:sns:us-east-2:000000000000:log-alerts-topic-2,
    ERROR_THRESHOLD=5,
    OUTPUT_BUCKET=processed-logs-bucket-2,
    APP_NAME=MyApp,
    ENVIRONMENT=production
  }"
```

### Verify Lambda Created

```bash
aws --endpoint-url=http://localhost:4566 lambda list-functions --query 'Functions[*].FunctionName'
```

## Step:-8

### Connect S3 → Lambda Trigger

#### Give S3 Permission to Invoke Lambda

```bash
aws --endpoint-url=http://localhost:4566 lambda add-permission \
  --function-name log-processor-node-2 \
  --statement-id s3-invoke-permission \
  --action lambda:InvokeFunction \
  --principal s3.amazonaws.com \
  --source-arn arn:aws:s3:::app-logs-bucket-2
```

#### Add S3 Trigger → Lambda

```bash
aws --endpoint-url=http://localhost:4566 s3api put-bucket-notification-configuration \
  --bucket app-logs-bucket-2 \
  --notification-configuration '{
    "LambdaFunctionConfigurations": [{
      "LambdaFunctionArn": "arn:aws:lambda:us-east-2:000000000000:function:log-processor-node-2",
      "Events": ["s3:ObjectCreated:*"],
      "Filter": {
        "Key": {
          "FilterRules": [{
            "Name": "suffix",
            "Value": ".log"
          }]
        }
      }
    }]
  }'
```
